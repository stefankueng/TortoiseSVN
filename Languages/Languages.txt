# List of languages supported by TortoiseSVN
#
# This list is basically CSV Format with the following rules:
# - Entries are separated by a Semicolon ';'. 
# - Strings which contain spaces should be enclosed in double quotes '"'
#   for better readability.
# - Unused languages are commented out by a hash mark '#'
#  
# This list is used for:
# - the Language pack builds
# - the Translation status report
#
# Handle with care. You might break something!
#
# ======================================================================
#
# Meaning of the columns:
# 1: Tag with multiple function
#    po/pot file (pot == 0, po <> 0)
#    Create language pack/download link (yes == 1, no <> 1)
#    Also affects sort order on status and download page. 
#    Sorted unsigned, by abs()
#    Recommended usage:
#    0 = pot file -> No download link, shown on translation status page as .po template
#    1 = po file  -> Create download link, shown on translation status page 
#    -1 = po file -> No download link, but shown on translation status page
#                    Use this to disable creation of a language pack
# 2: Country code. Used to display the flag. The [flagname].png has to match this column
# 3: WiX Language code. If the language doesn't have status 'released', put "en-us" there.
#    See here about the status: http://www.tramontana.co.hu/wix/loc/index.php
# 4: Locale ID for the language pack installer
# 5: Flag byte used to create links on different download pages
#    !!! Bit0 is the rightmost bit !!!
#    Bit0 = Spellchecker available
#    Bit1 = TortoiseSVN manual available
#    Bit2 = TortoiseMerge manual available
# 6: Full language name.
# 7: List of translators.
