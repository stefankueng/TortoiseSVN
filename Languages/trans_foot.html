Missing Languages

Would you like to translate TortoiseSVN into native language? Don't hesitate to download an empty catalog (trunk) and read these instructions, if you want to translate the current development version. Translate what you can and send us your translation. We will be glad to include it in the next TortoiseSVN release.

Download the current source dlls for the TortoiseSVN 1.0.8 translations and an empty catalog for 1.0.8, if you want to translate the last official release.

Download the ResText Utility to insert your translations into the dlls.
