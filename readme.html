Getting TortoiseSVN

If you want a copy of the source code, you have to use TortoiseSVN itself (or any other subversion client) to check out a working copy. Check it out from the following URL:

   http://svn.collab.net/repos/tortoisesvn/trunk

That will get you a new working copy, containing the latest sources. Read the build.txt file for instructions on building in the working copy.

You can browse the sourcecode online in HTML or using ViewCVS
