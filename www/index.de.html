TortoiseSVN 1.6.11 ist erhältlich

Dies ist ein Bugfix-Release, verlinkt mit Subversion 1.6.13.

Wir empfehlen so schnell wie möglich auf diese Version zu aktualisieren. Diese neue Version gibt es auf unserer Downloads Seite.

Bitte beachten Sie: wenn Sie von einer Version vor 1.6.10 aktualisieren müssen Sie eine Reparatur-Installation nach der eigenlichen Installation durchführen. Starten Sie den Installer einfach ein zweites Mal und klicken Sie auf "Repair" im Dialog.

Wenn Sie genau wissen wollen warum dies Notwendig ist lesen Sie bitte meinen früheren Beitrag (Englisch).
