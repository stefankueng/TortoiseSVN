TortoiseSVN 1.6.16 ist erhältlich

Dies ist ein Bugfix-Release, verlinkt mit Subversion 1.6.17.

Subversion 1.6.17 enthält unter anderem die folgenden sicherheitsrelevanten Korrekturen:

  • CVE-2011-1752
  • CVE-2011-1783
  • CVE-2011-1921

Hinweis: diese Sicherheitsprobleme betreffen nur Subversion Server, Clients sind nicht betroffen. Das bedeutet dass TortoiseSVN selbst nicht betroffen ist von diesen Problemen.

Wir empfehlen trotzdem so schnell wie möglich auf diese Version zu aktualisieren. Diese neue Version gibt es auf unserer Downloads Seite.

Bitte beachten Sie: wenn Sie von einer Version vor 1.6.10 aktualisieren müssen Sie eine Reparatur-Installation nach der eigenlichen Installation durchführen. Starten Sie den Installer einfach ein zweites Mal und klicken Sie auf "Repair" im Dialog.

Wenn Sie genau wissen wollen warum dies notwendig ist lesen Sie bitte meinen früheren Beitrag (Englisch).
