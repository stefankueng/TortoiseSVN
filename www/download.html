Misc Tools
SubWCRev.exe: a small command line tool which you can use in your build steps. It takes a file as an argument and replaces all "$WCREV$" strings with the revision of your working copy. This is useful if you want to include the Subversion revision in the version of your project. If you need an example, check out the TortoiseSVN source and/or have a look at the small help file. Note: If you have TortoiseSVN installed, you can find this tool in the TortoiseSVN installation folder! tigris.org
